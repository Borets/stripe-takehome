## stripe-takehome
This is take home from stripe that is currently in progress. 

## Run Locally

### 1. Clone repo

```
$ git clone git@github.com:/Borets/stripe-takehome.git
$ cd stripe-takehome
```

### 2. Run Backend

```
$ npm install
$ npm start
```

### 3. Run Frontend

```
# open new terminal
$ cd client/yaroslavs-shop/
$ npm install
$ npm start
```
